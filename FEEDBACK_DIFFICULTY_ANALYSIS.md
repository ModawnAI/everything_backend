# 피드백 항목별 난이도 분석

> **작성일**: 2025-12-08
> **총 피드백 항목**: 29개 (모바일 앱 21개 + 샵 어드민 8개)

---

## 난이도 기준

| 난이도 | 설명 | 예상 소요 시간 |
|--------|------|----------------|
| ⭐ 쉬움 | UI 변경만 필요, 백엔드 수정 없음 | 0.5일 이하 |
| ⭐⭐ 보통 | 간단한 API 추가 또는 기존 기능 확장 | 1-2일 |
| ⭐⭐⭐ 중간 | 새로운 DB 테이블 + API + 프론트엔드 | 2-3일 |
| ⭐⭐⭐⭐ 어려움 | 복잡한 비즈니스 로직, 여러 컴포넌트 연동 | 3-5일 |
| ⭐⭐⭐⭐⭐ 매우 어려움 | 외부 서비스 연동, 대규모 아키텍처 변경 | 5일 이상 |

## 모바일 앱 피드백

### 1. 홈 하단에 '내주변(지도)' 들어가기
**난이도**: ⭐⭐⭐⭐ 어려움

**설명**:
- 카카오맵 SDK 연동이 필요합니다
- 백엔드에 위치 기반 쿼리(Geospatial Query) 추가가 필요합니다
- 새로운 지도 페이지 UI 전체 개발이 필요합니다
- 사용자 위치 권한 처리 로직이 필요합니다

**필요 작업**:
- 백엔드: 위치 기반 샵 검색 API (`GET /api/shops/nearby`)
- 프론트: 카카오맵 컴포넌트, 마커 표시, 리스트/지도 토글
- 권한: GPS 위치 접근 권한 요청 플로우

### 3. 홈 화면 최상단에 포인트 내역 표시
**난이도**: ⭐⭐ 보통

**필요 작업**:

### 5. 홈 화면에 다양한 샵 섹션 추가
**난이도**: ⭐⭐⭐⭐ 어려움

**설명**:

**필요 작업**:
